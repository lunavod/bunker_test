С помощью этого плагина можно выполнять PHP код прямо в шаблонах с помощью тега {php}{/php} вот так:

  {php}
    echo "Hello, smarty3";
  {/php}

Т.к. с выходом smarty3, который используется в ЛС 1.0+, использование пхп в шаблонах больше недоступно.
